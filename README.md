# Интернет магазин на React, TypeScript, MobX

Приложение на сервере https://pavelstar.github.io/device-market-react-mobX/

## Стек технологий
- React
- TypeScript
- MobX
- SCSS

## Функционал
- Фильтрация 
- Поиск 
- Добавление в корзину
- Добавление в историю заказов
- Выбор цвета и объема памяти у продукта
